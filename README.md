# 🎯 AutoClicker - iOS 自动点击工具

可配置的 iOS 自动点击工具，支持 TrollFools 注入。

## ✨ 功能特性

- ✅ **自定义坐标点击** - 输入 X, Y 坐标精确点击
- ✅ **点击次数控制** - 设置固定次数或无限循环
- ✅ **点击频率调节** - 自定义点击间隔时间
- ✅ **按住时长设置** - 模拟短按或长按操作
- ✅ **悬浮按钮** - 快速访问配置界面
- ✅ **可拖动悬浮窗** - 随意调整位置
- ✅ **实时状态显示** - 查看当前点击进度
- ✅ **双击获取坐标** - 便捷的坐标获取方式
- ✅ **视觉反馈** - 点击位置显示红色圆圈

## 🚀 使用方法

### 1. 注入到应用

使用 TrollFools 将 `AutoClicker.dylib` 注入到目标应用。

### 2. 打开配置界面

打开应用后，点击屏幕右上角的 🎯 悬浮按钮。

### 3. 配置参数

- **点击坐标**: 输入 X, Y 坐标或双击获取
- **点击次数**: 设置固定次数或开启无限循环
- **点击间隔**: 设置每次点击的间隔时间（秒）
- **按住时长**: 设置每次点击按住的时长（秒）

### 4. 开始/停止

点击 "▶️ 开始" 开始自动点击，点击 "⏹ 停止" 停止。

## ⚠️ 注意事项

- 本工具仅供学习和研究使用
- 请勿用于违反应用服务条款的行为
- 游戏应用可能有反作弊检测，使用需谨慎
- 建议使用合理的点击间隔（0.5-2.0秒）

## 📝 使用场景

### ✅ 合理用途
- 自动关闭广告弹窗
- 自动签到
- 重复点击测试
- 辅助功能支持

### ⚠️ 不建议用途
- 游戏自动点击（可能封号）
- 刷量刷数据
- 自动抢购（违反公平性）

## 🔧 编译

### 要求

- Theos
- iOS SDK
- macOS / Linux

### 编译命令

```bash
make package
```

生成的 deb 包位于 `packages/` 目录。

提取 dylib：

```bash
dpkg-deb -x packages/*.deb extracted/
cp extracted/Library/MobileSubstrate/DynamicLibraries/AutoClicker.dylib .
```
